# Common Issues & Solutions
Last Updated: 2025-12-03T10:49:58.092-07:00

## SYNTAX ERRORS
**Issue**: "Unexpected end of input"
- **Cause**: Missing closing braces or unmatched brackets
- **Solution**: Check brace matching, use node -c filename.js to verify syntax

**Issue**: "Missing catch or finally after try"
- **Cause**: Try block without corresponding catch or finally
- **Solution**: Add catch block or remove unnecessary try

**Issue**: "Unexpected token 'catch'"
- **Cause**: Catch block without preceding try
- **Solution**: Remove orphaned catch or add proper try block

## GAME LOADING ISSUES
**Issue**: "createGameFunction is not defined"
- **Cause**: Syntax error preventing file from loading
- **Solution**: Check for duplicate functions, unmatched braces

**Issue**: Game returns undefined
- **Cause**: Function exits early due to error or missing return
- **Solution**: Add debug statements to trace execution path

## CONFIGURATION PROBLEMS
**Issue**: Config not loading
- **Cause**: JSON syntax errors or incorrect file paths
- **Solution**: Validate JSON, check file exists at expected path

## CANVAS ISSUES
**Issue**: Canvas not displaying
- **Cause**: Canvas not added to DOM or context not obtained
- **Solution**: Verify appendChild() call and getContext('2d') success
